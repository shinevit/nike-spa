**Prerequisites:**

- [Git](https://git-scm.com/)

**Installation**

Install the project dependencies:

```bash
npm install
```

**Running the Project**

```bash
npm start
```

**Open** [http://localhost:5173](http://localhost:5173) in your browser to view the app.
